第一次安装请先访问安装地址
http://你的域名/install/
按提示操作即可，做好给install目录写入权限
如有问题，请联系QQ540245094

后台地址：
http://你的域名/admin

支付接口配置：
在后台地址->系统设置->支付接口配置，在这里面填写阳光快付商户的key_id和key_token即可
如您还没有申请阳光快付接口，请先在https://pay.ygkpay.com/user/reg.php此链接中注册账号
登录商户中心，在商户入网处申请接口，获取key_id和key_token，再填入发卡网后台配置中即可
阳光快付提现手续费为：2%，除此之外别无其他费用。
如不会操作，可以联系QQ540245094辅助你
